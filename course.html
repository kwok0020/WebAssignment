2021 Fall Courses (level 1)



CST8300 Achieving Success in Changing Environments

This course is to let us assess our own aptitudes and strengths, and apply critical thinking and decision-making tools to help resolve some of the important issues in our complex society with its competing interests.

ENL 1813T Communications

Communication remains an essential skill sought by employers. We learn about the interview techniques and learn more regarding the job marketing. This lesson combin of lectures, group project as well as peer review. We practice writing, speaking, reading, listening and documenting by using technology to communication professionally. It helps us to develop strong communication skills in both educational and workplace environments.

CST8101 Computer Essentials

This course is about the essentials of computer software, hardware, and laptop management. We learn to configure the laptop environment, basic PC and troubleshoot problems. What's more, we learn how to create backups, install virus protection, and manage files in basic. By both in theory and through practical assignments to explore computer organization including basic numerical systems, functional hardware and software components needed to run programs.

CST8116 Introduction to Computer Programming

This course is the introductory knowledge of computer programming. We learn about problem analysis and design, using algorithms, pseudocode, flowcharts, UML Class Diagrams and testing. By both in theory and through practical assignments, we know more about the basic of Java programming by project.

CST8215 Database

This course is to learn the fundamentals of Relational Databases design using Entity Relation diagrams, and use SQL to create, modify and query a database. We will design and create databases that are maintainable, secure and adaptable to change in business requirements, using Normalization. Also, we learn the knowledge of Database Management System (DBMS) and how it works.

MAT8001C
Technical Mathematics for Computer Science

This course is about algebraic and transcendental functions is an essential prerequisite to Calculus. We learn about manipulate algebraic expressions, solve algebraic equations and linear systems and learn the properties of and graph algebraic and transcendental functions. What's more, investigate computer number systems in addition to Boolean algebra and logic to help solve problems involving computer systems.
